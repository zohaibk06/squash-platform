# Prototype archive

Snapshots of `prototype.html` at meaningful milestones, kept verbatim for
reference so we can revisit earlier directions without digging through git
history. The canonical, latest prototype lives at the repo root as
`prototype.html` — these are frozen copies.

| File | From commit | What it is |
|------|-------------|------------|
| `01-five-direction-explorer.html` | [`df49928`](../../../commit/df49928) | The original 5-direction visual explorer: **Court-Side**, **Broadsheet**, **Command Deck**, **Locker Room**, **Logbook** — each with coach/athlete/parent persona toggles. Disconnected visual screens, no real navigation. Reviewed with Ahad on 2026-05-10 evening (PRD §11). |
| `02-comprehensive-arpro-journal-clay.html` | [`6f3fdae`](../../../commit/6f3fdae) | First comprehensive coach + athlete end-to-end clickable rebuild on the Locker Room + Logbook hybrid IA. Three themes: **arpro**, **journal** (light editorial — Fraunces + Hanken Grotesk), **clay** (soft + friendly — Bricolage Grotesque + terracotta). Useful if we want to revisit those two abandoned aesthetic directions. |
| `03-arpro-logbook-pulse-desktop.html` | [`e26d5f1`](../../../commit/e26d5f1) | The current theme set settled — **arpro**, **logbook** (Clash Display + JetBrains Mono + Archivo), **pulse** (oxblood + Clash + Instrument Serif) — plus the **Phone / Desktop layout toggle**. Builder is still the simple drill-card version (no editable section blocks yet); no scheduling wizard, no three-type assignments, no inbox filters. Useful as the cleanest "before Stuti walkthrough" snapshot. |

Open each file directly in a browser — they are fully self-contained static
HTML with no build step.
